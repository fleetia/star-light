# NavigationButton

페이지 간 이동을 위한 방향 버튼 컴포넌트입니다. 좌우 화살표를 표시하여 이전/다음 탐색을 제공합니다.

## 접근성

- `button` 역할을 가지며 키보드 `Enter` 및 `Space`로 활성화할 수 있습니다.
- 방향에 따라 적절한 `aria-label`을 제공해야 합니다.
- 포커스 시 시각적 표시가 나타납니다.
