# PositionGrid

3x3 격자에서 위치를 선택할 수 있는 컴포넌트입니다. 요소 배치 방향을 직관적으로 지정할 수 있습니다.

## 접근성

- 각 위치가 `button` 역할을 가지며 키보드로 선택할 수 있습니다.
- 선택된 위치에 `aria-pressed="true"` 속성이 적용됩니다.
- `Tab`으로 격자 내 버튼 간 이동이 가능합니다.
