# CollapsibleSection

접을 수 있는 섹션 컴포넌트입니다. 제목을 클릭하면 내부 콘텐츠가 펼쳐지거나 접힙니다.

## 접근성

- 토글 버튼에 `aria-expanded` 속성이 현재 상태를 반영합니다.
- `Enter` 및 `Space` 키로 섹션을 펼치거나 접을 수 있습니다.
- 스크린 리더가 섹션의 열림/닫힘 상태를 인식합니다.
