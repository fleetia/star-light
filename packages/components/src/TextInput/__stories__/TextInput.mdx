## 접근성

- `label` 요소가 입력 필드와 `htmlFor`/`id`로 연결됩니다.
- 에러 상태에서 `aria-invalid="true"`와 `aria-describedby`로 에러 메시지가 연결됩니다.
- 키보드로 입력 및 탐색이 가능합니다.
- 필수 입력인 경우 `aria-required="true"`를 지정합니다.
