# ConfirmDialog

사용자에게 확인을 요청하는 대화상자 컴포넌트입니다. 확인/취소 버튼과 메시지를 표시합니다.

## 접근성

- `alertdialog` 역할이 적용되어 긴급한 상호작용임을 알립니다.
- 모달과 동일하게 포커스 트랩이 동작합니다.
- `Escape` 키로 취소할 수 있습니다.
- 확인 버튼에 기본 포커스가 위치하지 않도록 주의합니다 (파괴적 동작 방지).
