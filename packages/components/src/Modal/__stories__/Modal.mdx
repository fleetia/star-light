# Modal

콘텐츠 위에 오버레이로 표시되는 모달 대화상자 컴포넌트입니다. 제목, 크기 옵션을 지원합니다.

## 접근성

- `dialog` 역할과 `aria-modal="true"`가 적용됩니다.
- 모달이 열리면 포커스가 모달 내부로 이동하고, 포커스 트랩이 동작합니다.
- `Escape` 키로 모달을 닫을 수 있습니다.
- 닫힐 때 포커스가 이전 요소로 복원됩니다.
