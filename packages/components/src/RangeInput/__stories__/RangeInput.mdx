# RangeInput

슬라이더 형태의 범위 입력 컴포넌트입니다. 라벨과 현재 값을 함께 표시할 수 있습니다.

## 접근성

- `slider` 역할을 가지며 `aria-valuemin`, `aria-valuemax`, `aria-valuenow`가 설정됩니다.
- 화살표 키(`←`, `→`)로 값을 조절할 수 있습니다.
- 라벨이 슬라이더와 연결되어 스크린 리더가 용도를 인식합니다.
