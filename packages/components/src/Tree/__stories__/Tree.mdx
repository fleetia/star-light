# Tree

계층적 데이터를 트리 구조로 표시하는 컴포넌트입니다. 노드 펼침/접기, 가시성 토글, 드래그 재정렬을 지원합니다.

## 접근성

- `tree` 역할을 가지며 각 노드는 `treeitem` 역할을 가집니다.
- 화살표 키(`↑`, `↓`)로 노드 간 이동, `←`로 접기, `→`로 펼치기가 가능합니다.
- 펼침/접힘 상태가 `aria-expanded`로 표시됩니다.
- 가시성 토글 버튼에 적절한 `aria-label`이 제공됩니다.
