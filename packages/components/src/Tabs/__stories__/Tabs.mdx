## 접근성

- `tablist` 역할을 가지며 각 탭은 `tab` 역할을 가집니다.
- 화살표 키(`←`, `→`)로 탭 간 이동이 가능합니다.
- 활성 탭에 `aria-selected="true"` 속성이 적용됩니다.
- 탭 패널에 `tabpanel` 역할과 해당 탭과의 연결이 설정됩니다.
