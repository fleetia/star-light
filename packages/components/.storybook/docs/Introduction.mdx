# Launchpad Design System

`@launchpad/components`는 Launchpad 브라우저 확장 프로그램을 위한 디자인 시스템입니다.

## 설치

```bash
pnpm add @launchpad/components
```

## 테마 시스템

모든 컴포넌트는 CSS 변수 기반 테마 시스템을 사용합니다. 런타임에 `setCSSVariable` 또는 `setCSSVariables`를 호출하여 테마를 변경할 수 있습니다.

```tsx
import { setCSSVariables, darkTheme } from "@launchpad/components";

// 다크 테마 적용
setCSSVariables({
  "--c-accent": darkTheme.accent,
  "--c-accent-text": darkTheme.accentText,
  "--c-surface": darkTheme.surface,
  "--c-text": darkTheme.text,
  "--c-border": darkTheme.border,
  "--c-hover-bg": darkTheme.hoverBg,
  "--c-hover-text": darkTheme.hoverText,
  "--c-muted": darkTheme.muted
});
```

## CSS 변수 레퍼런스

| 변수명            | 설명               | 기본값    |
| ----------------- | ------------------ | --------- |
| `--em`            | 기본 단위          | `16px`    |
| `--icon-size`     | 아이콘 크기        | `32px`    |
| `--gap`           | 기본 간격          | `1em`     |
| `--c-accent`      | 강조 색상          | `#000000` |
| `--c-accent-text` | 강조 텍스트 색상   | `#ffffff` |
| `--c-surface`     | 배경 색상          | `#ffffff` |
| `--c-text`        | 텍스트 색상        | `#000000` |
| `--c-border`      | 테두리 색상        | `#000000` |
| `--c-hover-bg`    | 호버 배경 색상     | `#000000` |
| `--c-hover-text`  | 호버 텍스트 색상   | `#ffffff` |
| `--c-muted`       | 비활성 텍스트 색상 | `#999999` |

## 컴포넌트

### 레이아웃

- **Modal** — 오버레이 모달 컨테이너
- **Box** — 테두리 있는 그룹 컨테이너
- **Sidebar** — 사이드바 레이아웃
- **CollapsibleSection** — 접기/펼치기 섹션
- **Tabs** — 탭 네비게이션

### 네비게이션

- **Breadcrumb** — 경로 탐색
- **NavigationButton** — 이전/다음 버튼
- **CardPagination** — 페이지네이션
- **Tree** — 재귀 트리 구조

### 입력

- **Button** — 범용 버튼 (primary, secondary, ghost, danger)
- **TextInput** — 텍스트 입력
- **Checkbox** — 체크박스
- **Toggle** — 토글 스위치
- **RangeInput** — 슬라이더
- **ColorPicker** — 색상 선택기
- **PositionGrid** — 위치 선택 그리드

### 표시

- **Badge** — 상태 뱃지
- **ContextMenu** — 우클릭 메뉴
- **ConfirmDialog** — 확인 다이얼로그
- **IconButton** — 아이콘 버튼
- **ColorRow** — 색상 행

### 아이콘

- ChevronLeftIcon, ChevronRightIcon
- TriangleUpIcon, TriangleDownIcon
- GearIcon, FolderIcon, EyeIcon, DragHandleIcon

## 기술 스택

- **React 18** — UI 프레임워크
- **vanilla-extract** — 타입 안전 CSS-in-TS
- **Storybook 10** — 컴포넌트 문서화
- **Vitest** — 테스트
