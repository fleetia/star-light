# Changelog

이 파일은 크보니트의 주요 변경 사항을 기록합니다.
형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.1.0/)을 따르고,
버전 규칙은 [Semantic Versioning](https://semver.org/lang/ko/)을 따릅니다.

## [1.2.0] - 2026-04-29

### Added

- **취소 경기(우천취소 포함) 지원**
  - 옵션 탭에서 "취소" 색상 지정 (홈/원정 분리 가능)
  - "취소 경기 N줄 (0=숨김)" 입력으로 목도리에 칸을 추가하거나 제외
  - 가로 미리보기·세로 패턴·뜨개 가이드·단수 카운터 모두에 반영
  - 통계 영역에 취소 경기 수 표시 (`| 취소 N`)
  - 수동 추가 모달에서 "취소 경기로 추가" 체크박스 지원
  - 데이터 스크래퍼가 KBO 사이트의 "취소" 표기를 인식해 `status: "cancelled"` 필드로 저장

- **옵션 탭 분리**
  - 시즌·팀·시리즈·색상·줄 수 설정을 새 `옵션` 탭으로 모음
  - 첫 방문자는 `옵션` 탭이 기본으로 열려 셋업이 자연스러움
  - 기존 사용자는 마지막으로 보던 탭이 그대로 유지됨
  - 가로 미리보기와 "+ 경기 결과 미리 추가" 버튼은 탭 위에 항상 표시되어, 옵션을 바꾸면 위쪽에서 즉시 반영 확인 가능

- **PWA 자동 업데이트**
  - 설치된 PWA를 사용 중인 경우, 새 버전이 배포되면 다음 실행 시 자동으로 새 버전이 적용됨
