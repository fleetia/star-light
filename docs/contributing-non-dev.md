# 비개발자 기여 가이드

코드를 작성하지 않아도 이 프로젝트에 기여할 수 있습니다!

## 버그 리포트

사용 중 문제를 발견했다면 [GitHub Issues](../../issues)에 이슈를 남겨주세요.

좋은 버그 리포트에 포함되면 좋은 내용:

- 어떤 문제가 발생했는지
- 어떻게 하면 문제를 재현할 수 있는지
- 기대했던 동작과 실제 동작
- 브라우저/OS 정보
- 스크린샷 (있다면)

## 기능 제안

새로운 기능이나 개선 아이디어가 있다면 이슈로 남겨주세요. "이런 게 있으면 좋겠다" 수준의 가벼운 제안도 환영합니다.

## UX 피드백

사용하면서 느낀 불편함이나 개선점을 공유해주세요:

- 직관적이지 않은 UI
- 헷갈리는 용어나 설명
- 접근성 문제
- 성능 문제 (느림, 버벅임 등)

## 번역

이 프로젝트는 일부 다국어를 지원합니다. 번역에 참여하고 싶다면 이슈로 알려주세요.

## 문서 개선

오타, 잘못된 설명, 부족한 내용 등을 발견했다면 직접 PR을 보내거나 이슈로 알려주세요. 문서 수정은 코드 지식 없이도 가능합니다.
